# Stack Problem Solving (C++)

This repository contains solutions to various **stack-based problems** implemented in **C++**.
It focuses on understanding the **stack data structure** and its application in solving algorithmic and coding interview problems.

## 📌 Topics Covered

* Stack fundamentals
* Valid Parentheses
* Next Greater Element
* Expression Evaluation
* Monotonic Stack Problems
* Balanced Brackets
* Other common stack interview problems

## 🎯 Purpose

The goal of this repository is to:

* Practice stack-based problem solving
* Strengthen **data structure concepts**
* Improve **logical thinking and coding skills**
* Prepare for **coding interviews and competitive programming**

## 🛠️ Technologies Used

* C++
* Standard Template Library (STL)
* Stack data structure

## 📂 Structure

Each file contains a solution to a specific stack-related problem along with clean and readable C++ code.

## 🚀 How to Run

1. Clone the repository
2. Open the file in any C++ compiler or IDE
3. Compile and run the program

```bash
g++ filename.cpp
./a.out
```
